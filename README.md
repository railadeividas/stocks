# Stocks Portfolio Tracker

Asmeninis akcijų portfelio sekimo įrankis. GitHub Pages aplikacija.

**URL:** https://railadeividas.github.io/stocks/

## Funkcionalumas

- ✅ Pirkimų įrašymas (simbolis, kiekis, kaina, data, pastabos)
- ✅ Pardavimų įrašymas
- ✅ Portfelio suvestinė (investuota suma, likutis, P/L)
- ✅ Vidutinė vieneto kaina kiekvienam stock'ui
- ✅ Realizuotas/neralizuotas pelnas/nuotolis
- ✅ Gyvos kainos iš Yahoo Finance
- ✅ Valiutų konversija (USD → EUR pagal gyvą kursą)
- ✅ Transakcijų istorija su paieška/filtravimu
- ✅ Redagavimas/trynimas
- ✅ CSV/JSON eksportas
- ✅ Duomenys išlieka naršyklėje (localStorage)
- ✅ Lietuviška sąsaja

## Kaip naudotis

1. Atsidaryk: https://railadeividas.github.io/stocks/
2. Pridėk pirkimą: „Pridėti pirkimą" → užpildyk formą
3. Pridėk pardavimą: „Pridėti pardavimą" → nurodyk simbolį, kiekį, kainą
4. Stebėk suvestinę pagrindiniame puslapyje
